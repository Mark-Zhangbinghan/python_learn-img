### git基础指令

| 命令                       | 说明                                                         |
| -------------------------- | ------------------------------------------------------------ |
| git init                   | 创建一个github的仓库                                         |
| git clone                  | 从github上提取项目出来                                       |
| cd "文件名"                | 打开该文件夹                                                 |
| git add .                  | 把文件夹下面的文件全部添加进入仓库                           |
| git commit  -m  "提交信息" | 记录修改内容                                                 |
| git push -u origin "分支"  | 此操作目的是把本地仓库push到github上面，此步骤需要你输入帐号和密码 |
